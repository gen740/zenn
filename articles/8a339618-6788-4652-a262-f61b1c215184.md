# Tex

## コード環境

* Minted

```latex
\usepackage{minted}
\setminted{style=lovelace,fontsize=\small,numbers=left}
...

\begin{minted}{cpp}
#include <iostream>

int main {
  std::cout << "Hello World" << std::endl;
}
\end{minted}
```

* inline code

```latex
\usepackage{listings}

\lstinline{inline}
\codeword{here}
```



# Hyperlink

```latex
% プリアンブル
\usepackage{hyperref}
\hypersetup{
  colorlinks = true,
  linkcolor = blue,
  urlcolor = blue,
  citecolor = blue,
  anchorcolor = blue,
  filecolor = blue,
  menucolor = blue,
  runcolor = blue,
  linkbordercolor = {0 1 1},
  urlbordercolor = {0 1 1},
  pdfborderstyle={/S/U/W 1} % 下線スタイル
}

\href{url}{テキスト} % URL に対する hyperlink

% ターゲットの設定
\hypertarget{label_name}{ここにターゲットテキスト}

% リンクの作成
\hyperlink{label_name}{ここをクリックしてください}
```



# Image と Minipage

```latex


\begin{minipage}[b]{0.45\linewidth}
  \includegraphics[width=\linewidth]{image1.jpg}
  \caption{最初の画像}
  \label{fig:image1}
\end{minipage}
```

# TOC のフォーマット

```latex
\usepackage{tocloft}

\setcounter{tocdepth}{3}
\cftsetindents{section}{0em}{2.5em}
\cftsetindents{subsection}{2.5em}{3.3em}
\cftsetindents{subsubsection}{5.8em}{4.1em}
{\small \tableofcontents}
```
