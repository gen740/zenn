# 手順

1. cmake をクローン

    ```bash
    git clone https://github.com/Kitware/CMake.git
    ```
    ```bash
    gh repo clone Kitware/CMake
    ```
1. boostwrap を実行

    ```bash
    ./bootstrap --generator=Ninja --prefix=$HOME/.local
    ```
1. ビルドしてインストール

    ```bash
    ninja && ninja install
    ```
